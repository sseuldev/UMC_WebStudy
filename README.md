---
- login.html & style.css & login.js 를 활용하여 페이지 구현

  - 비밀번호를 5회 이상 틀리는 경우 : *"비밀번호를 5회 이상 틀리셨습니다. 비밀번호 찾기를 권장드립니다."*

### 3. React 시작하기 (React_환경설정)
---
- react 앱 시작하기
```
npx create-react-app (파일명)
```
- 경로 변경 (change directory)
```
cd (파일명)
```
- 앱 실행 -> npm의 조금 더 개선된 버전 - 더 빠르다
```
yarn start
```
